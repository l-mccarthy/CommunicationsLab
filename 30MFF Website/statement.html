What filming techniques did we use?

  • Directing

    Helping stage the actors' movements so that it seems natural, bringing in a documentary-like feel.

  • Cinematography

    Framing each shot to look like the last.

  • Editing

    We cropped the videos to look similar to aid a match cut so the action and flow within the frame move smoothly in between shots and hides the cut.

  • Music

    The use of a simple pop/disco song helps the editor find the rhythm of the cut to structure each edit so that everyone is on beat, creating continuity.
